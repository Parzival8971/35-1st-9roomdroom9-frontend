### 🐥 앞뒤가 똑같은 구방문방구!

- 앞뒤가 똑같은 구방문방구를 팀명으로 팀원들의 물건을 재미있게 판매하는 서비스 홈페이지 구현했어요!

- 외부 라이브러리를 최대한 활용하지 않고 초기세팅부터 직접 구현했으며, 백엔드와 API까지하여 기능을 구현했어요!
