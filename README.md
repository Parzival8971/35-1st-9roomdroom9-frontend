#### 메인화면

- 메인 슬라이드 이미지와 Nav,Footer로 디자인을 완성했습니다.
- component를 사용하여 코드의 가독성과 재활용성을 높였다.
- map함수로 메인페이지의 반복적이고 동일한 레이아웃을 데이터를 일일이 적지않도록 하였다.
